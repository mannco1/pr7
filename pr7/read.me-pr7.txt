# Программа для решения уравнений 1-го и 2-го типа

# Класс type1

Класс type1 предназначен для уравнений вида 0 = 0, которые имеют бесконечное множество решений.

# Методы класса type1

- type1() - конструктор по умолчанию, не принимает никаких параметров.
- void Get_answer() - метод, который выводит на экран сообщение о том, что решением уравнения является любое х.
- void show() - метод, который выводит на экран уравнение вида 0 = 0.

# Класс type2

Класс type2 предназначен для уравнений вида Ax^2 = 0, которые имеют одно решение х = 0 при A != 0.

# Поля класса type2

- double A - коэффициент А уравнения.

# Методы класса type2

- type2(double a1) - конструктор с параметром, который принимает значение коэффициента А и присваивает его полю A.
- void Get_answer() - метод, который выводит на экран сообщение о том, что решением уравнения является х = 0.
- void show() - метод, который выводит на экран уравнение вида Ax^2 = 0 с заданным коэффициентом А.
